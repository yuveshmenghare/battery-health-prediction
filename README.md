# 🔋 Battery Health Predictor (Streamlit App)

This project predicts smartphone battery health using machine learning.

## Features
- Predicts battery health in real-time
- Streamlit-based interactive UI
- Uses Random Forest model
- Feature scaling with StandardScaler

## How to Run

```bash
pip install -r requirements.txt
streamlit run app.py

```
Inputs
Device age
Charging cycles
Battery temperature
Usage patterns


Output
Predicted battery health (%)
Recommended action
